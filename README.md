This is my submission for CyberDefenders evaluation task: Cybersecurity Certification Roadmap Chart Component

## Setup Instructions

First, clone the repository: https://github.com/ehabhosam/cyberdefenders-chart.git

Then, navigate to the project directory and install the dependencies:

```bash
cd cyberdefenders-chart
npm install 
```

To run the development server, use the following command:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
